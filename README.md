# 📈 ProStock: Full-Stack Financial Analytics Dashboard

A high-performance stock visualization tool using a decoupled **FastAPI** backend and an interactive **Streamlit** frontend.

## 🌟 Features
* **Asynchronous API:** Built with FastAPI for high-speed data serving.
* **Interactive Candlestick Charts:** Powered by Plotly for professional-grade analysis.
* **Live Metadata:** Fetches sector info, business summaries, and real-time metrics.
* **Decoupled Architecture:** Frontend and Backend communicate via REST API.
